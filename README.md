# 🌱 EcoSort AI – Waste Segregation Assistant

EcoSort AI is an AI-powered waste segregation assistant designed to promote sustainable waste management practices. The application helps users identify waste items through text input or image uploads, classify them into appropriate waste categories, and receive disposal recommendations along with environmental insights.

Built with a focus on sustainability and aligned with **United Nations Sustainable Development Goal (SDG) 12 – Responsible Consumption and Production**, EcoSort AI aims to educate users about responsible waste disposal and encourage environmentally conscious behavior.

---

🔗 **Demo Link:** [https://ecosort-ai-frfw.onrender.com]



## 🎯 Problem Statement

Improper waste segregation is one of the major contributors to environmental pollution and inefficient recycling processes. Many people struggle to identify the correct disposal method for various waste items.

EcoSort AI addresses this issue by providing an intelligent assistant that helps users classify waste accurately and promotes sustainable disposal practices.

---

## 💡 Solution

EcoSort AI leverages Artificial Intelligence concepts to:

- Identify waste items from text or uploaded images
- Categorize waste into appropriate waste classes
- Provide disposal recommendations
- Educate users about environmental impact
- Promote recycling awareness
- Encourage sustainable waste management

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/jaiswalshourya33/EcoSort-AI-.git
```

### Navigate to Project

```bash
cd EcoSort-AI-
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env.local` file:

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

### Run Development Server

```bash
npm run dev
```

Application will run at:

```text
http://localhost:5173
```
